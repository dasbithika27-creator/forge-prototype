# FORGE — Sustainable Procurement Platform
FORGE — AI Procurement Intelligence Platform
AI-driven procurement system designed to transform how enterprises discover suppliers, optimise pricing, and track Scope 3 emissions across complex supply chains.
> An interactive prototype demonstrating AI-driven procurement intelligence for large-scale infrastructure projects.

**Live Demo →https://dasbithika27-creator.github.io/forge-prototype/

---

## What It Does

FORGE is a concept product that reimagines how infrastructure companies manage procurement — combining supplier intelligence, carbon tracking, and AI-driven recommendations in a single platform.

Built as a portfolio prototype to demonstrate product thinking across B2B commerce, sustainability, and AI-assisted decision-making.

---

## Features

| Module | Description |
|---|---|
| **Dashboard** | Spend overview, Scope 3 tracker, and live alerts |
| **Supplier Marketplace** | 2,497 suppliers scored on price, carbon grade, and supply chain risk |
| **Live Reverse Auction** | Real-time bidding with countdown timer and live price updates |
| **AI Recommendations** | Supplier suggestions ranked by price, carbon intensity, and delivery reliability |
| **Price Trends** | Commodity price tracking with AI-generated forward forecast |
| **Demand Forecasting** | 6-month materials demand prediction by project |
| **Project Budgets** | Budget vs spend and carbon P&L per project |
| **Approval Workflow** | Multi-level PO sign-off (L1/L2/L3) |
| **Sustainability Report** | Scope 1, 2 and 3 emissions progress, supplier carbon league table, carbon vs cost trade-off analysis |

## Built With

- React 18 (via CDN, no build step required)
- Recharts for data visualisation
- Vanilla CSS with inline styles
- Fully self-contained — single `index.html` file

## Running Locally

No installation needed. Just open `index.html` in any modern browser.

```bash

cd forge-prototype
open index.html
```
